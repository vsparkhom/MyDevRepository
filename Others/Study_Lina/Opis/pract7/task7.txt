Практическое задание к теме №8
Тема: «Создание XML и XML Schema документов»
 

Задание.

1. Согласно заданию для своего варианта создать XML документ, а именно:
   - создать структуру XML-документа;
   - определить теги документа и их атрибуты;
   - наполнить документ содержимым;
   - не забудьте указать пролог, содержащий версию языка XML, кодировки документа.
2. Проверить правильность созданного документа в браузере.
3. Для созданного XML документа создайте соответствующий ему XML Schema документ.
4. С помощью онлайн валидатора http://www.corefiling.com/opensource/schemaValidate.html проверьте соответствие XML Schema её XML документу.
5. Добавить в отчет таблицу с описанием тегов и скриншот результатов валидации.
