Практическое задание к теме №5

Тема: «Создание простых скриптов с помощью JavaScript»

Требования к оформлению работы:

1. Результатом выполнения работы является отчет, который должен содержать решение поставленных задач.
2. Отчет следует сопроводить как текстовыми комментариями, так и необходимыми скриншотами, фрагментами кода.

Задание                 

1. Создать HTML страницу с формой для расчета искомой величины, согласно варианту.
2. Предусмотреть поля для ввода входных данных и кнопки для расчета и очистки полей.
3. С помощью скрипта проверить заполнение обязательных полей и валидность входных данных. Если данные введены неверно или поле пустое — выводить сообщение в модальное окно.
4. Результат расчета выводить в блок после текстовых полей (для четных вариантов) или текстовое поле (для нечетных вариантов) в формате: Текст + Результат.
